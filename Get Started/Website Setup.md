# [MindStone](https://mindstone.tuancao.me/)
## Recommended Platform
- Windows
- macOS
- Linux
- iOS
- iPadOS
- Android
- ChromeOS
## Setup Prerequisites
Create an account for the following services:
- [GitHub](https://www.github.com/join) 
- Repl.it
- [Netlify](https://app.netlify.com/signup)
## Steps
1. Fork the original repo into your GitHub Account.
2. Import the GitHub repo you forked recently on Repl.it.
3. Select Node.js as your language.
4. Run `npm install -g npm@9.8.1` to update Node.js.
5. Input `npm install --global yarn` to install yarn package, and then `yarn add yarn` to check missing dependences of yarn.
6. Run `yarn && yarn run dev` to just verify that there are no errors, there is no need to go `http://localhost:3000/` as you cannot access it when running on Repl.it.
7. Go to “Git” from “Tools”.
8. Click on the Current branch, by default it is named as "main".
9. You are required to input manually a summary first, before pressing “Stage and commit all changes”.
10. Go now to Netlify and import the same GitHub repo you own.
11. Make sure that the setup settings are exactly as the one provided below:
	- base directory:
	- package directory:
	- build command: **next build**
	- Publish directory: **.next**
