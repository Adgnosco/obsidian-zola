# Differential calculus

## Demostration
- Constant potential function
	1. f(x) = k, k ∈ R
	2. x ≠ k
	3. f(x) => f'(x), k => 0
	4. ==f'(x) = 0==
- Identity potential fuction
	1. f(x) = x, x = x¹
	2. f(x) = x¹
	3. f(x) => f'(x), x => n · xⁿ⁻¹
	4. f'(x) = nxⁿ⁻¹
	5. f'(x) = 1 · x¹⁻¹
	6. f'(x) = 1 · x/x
	7. f'(x) = 1 · 1
	8. ==f'(x) = 1==
